NEW ENDPOINTS ADDED:

1. POST /api/admin/add-item
   Upload a new thrift item
   Body: { admin_id, item_name, category, eco_points_cost, image_url, hub_id }
   Returns: { success, item_id, message }

2. GET /api/admin/inventory
   List admin's items
   Query: ?admin_id=1&hub_id=1
   Returns: Array of items

3. GET /api/admin/messages
   Get contact messages
   Query: ?admin_id=1&is_read=false
   Returns: { messages, total, limit, offset }

4. GET /api/admin/messages/:messageId
   Get single message
   Query: ?admin_id=1
   Returns: Message object

5. PATCH /api/admin/messages/:messageId/read
   Mark message as read
   Body: { admin_id }
   Returns: { success }

6. POST /api/contact-admin
   Submit contact form (PUBLIC)
   Body: { name, email, phone, organization, message }
   Returns: { success, message_id, received_at }

7. GET /api/impact/dashboard
   Get environmental impact data (ENHANCED)
   Query: ?user_id=1&hub_id=1
   Returns: Real data from waste_activities table
